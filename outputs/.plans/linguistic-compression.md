# Plan: Linguistic Theory and Constructed Languages as Compression Systems

## Topic
How do constructed languages function as compression systems relative to natural language, and what does information theory predict about their trade-offs?

## Key Sub-Questions
1. What is the information-theoretic baseline for natural language (entropy, redundancy, Zipf's law, universal information rate)?
2. How do vocabulary-minimalist conlangs (Toki Pona, Basic English) achieve compression, and what disambiguation costs do they impose?
3. How do precision-maximalist conlangs (Lojban, Ithkuil) handle the ambiguity-verbosity trade-off?
4. What does structural restriction (E-Prime) accomplish as a compression mechanism?
5. How do international auxiliary languages (Interlingua) fit into the compression framework?
6. What is the theoretical synthesis — the ambiguity-verbosity trade-off and Uniform Information Density?

## Agent Assignments
- Agent 1 (info-theory): Shannon entropy, Zipf's law, Coupé 2019 universal rate, UID hypothesis
- Agent 2 (minimalist-conlangs): Toki Pona, Basic English, community data, disambiguation cost
- Agent 3 (precision-conlangs): Lojban, Ithkuil, E-Prime, predicate logic, morphological density

## Acceptance Criteria
- Every factual claim has an inline citation
- Conflicts and uncertainties are explicitly flagged
- Comparison table includes all 7 systems
- Provenance file tracks all sources and review verdict
