<objective>
Add Thompson sampling to memory selection so that ranked candidates are re-ordered
probabilistically before token budget packing. This replaces deterministic ordering
with explore/exploit behavior: well-used memories are still favored, but cold-start
and underexplored memories get a fair chance via the Beta(1,1) prior.

Purpose: Foundation requirement FOUND-03. Enables the system to discover whether
less-used memories are actually valuable, rather than always surfacing the same
highest-ranked ones.

Output: `_thompson_rerank` method on RetrievalEngine, wired into both the hybrid
and static retrieval paths, guarded by the `thompson_sampling` feature flag.
</objective>

    ```python
    def _thompson_rerank(self, memories: list) -> list:
        """Re-rank memories using Thompson sampling over Beta(usage+1, unused+1)."""
        import random
        scored = []
        for mem in memories:
            a = (mem.usage_count or 0) + 1
            b = max((mem.injection_count or 0) - (mem.usage_count or 0), 0) + 1
            sample = random.betavariate(a, b)
            scored.append((sample, mem))
        scored.sort(key=lambda x: x[0], reverse=True)
        return [mem for _, mem in scored]
    ```
