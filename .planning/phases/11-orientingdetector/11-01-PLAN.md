<objective>
Create a standalone rule-based OrientingDetector that identifies high-signal moments in text — user corrections, explicit emphasis, error spikes, and pacing breaks — and returns structured results with an importance boost score.

Purpose: The OrientingDetector is the first component of the Observation Quality pipeline (Phases 11-14). It catches the moments that matter most — when the user corrects Claude, stresses something important, hits a wall of errors, or shifts gears abruptly. These are the moments biological memory systems flag via the orienting response (involuntary "what is it?" reflex). By flagging them before consolidation, we ensure the LLM consolidator prioritizes them (Phase 14 will wire this in).

Output: `core/orienting.py` module with `OrientingDetector`, `OrientingResult`, and `OrientingSignal` classes. Feature flag in `core/flags.py`. Full test coverage in `tests/test_orienting.py`.
</objective>

    ```python
    from dataclasses import dataclass, field

    @dataclass
    class OrientingSignal:
        signal_type: str       # "correction", "emphasis", "error_spike", "pacing_break"
        confidence: float      # 0.0 to 1.0
        matched_text: str      # the text fragment that triggered
        importance_boost: float # how much to boost importance

    @dataclass
    class OrientingResult:
        signals: list[OrientingSignal] = field(default_factory=list)
        importance_boost: float = 0.0  # max boost across all signals

        @property
        def has_signals(self) -> bool:
            return len(self.signals) > 0
    ```

    2. **OrientingDetector class:**
    ```python
    class OrientingDetector:
        # Compiled regex patterns for each signal type
        CORRECTION_PATTERNS = [
            # Word-boundary patterns for correction language
            # "no" needs careful handling — "no, that's wrong" yes, "no problem" maybe not
            # Use phrase patterns for higher precision
            r"\b(no[,.]?\s+that'?s?\s+(wrong|not|incorrect))",
            r"\b(actually[,\s])",  # "actually" at word start followed by comma/space
            r"\b(not that[,\s])",
            r"\b(i said\b)",
            r"\b(that'?s\s+not\s+(right|correct|what))",
            r"\b(wrong[,.\s])",
            r"\b(incorrect\b)",
        ]

        EMPHASIS_PATTERNS = [
            r"\b(remember\s+this)",
            r"\b(always\s+\w+)",    # "always use", "always run"
            r"\b(never\s+\w+)",     # "never use", "never skip"
            r"\b(important\b)",
            r"\b(critical\b)",
            r"\b(don'?t\s+forget)",
            r"\b(make\s+sure\b)",
        ]

        ERROR_INDICATORS = [
            r"(?i)(error:|Error:)",
            r"(?i)(traceback\s*\(most recent)",
            r"(?i)(failed:|FAILED)",
            r"(?i)(exception:)",
            r"(?i)(ModuleNotFoundError|ImportError|TypeError|ValueError|KeyError|AttributeError|NameError|RuntimeError|AssertionError)",
        ]

        ERROR_SPIKE_THRESHOLD = 3  # minimum error indicators for spike

        IMPORTANCE_BOOSTS = {
            "correction": 0.3,
            "emphasis": 0.2,
            "error_spike": 0.2,
            "pacing_break": 0.1,
        }

        PACING_BREAK_RATIO = 0.4  # message < 40% of recent average = pacing break
    ```

    3. **detect() method:**
    ```python
    def detect(self, text: str, message_lengths: list[int] | None = None) -> OrientingResult:
        from .flags import get_flag

        if not get_flag("orienting_detector"):
            return OrientingResult()

        if not text:
            return OrientingResult()

        signals = []

        # Check correction patterns (case-insensitive)
        for pattern in self.CORRECTION_PATTERNS:
            match = re.search(pattern, text, re.IGNORECASE)
            if match:
                signals.append(OrientingSignal(
                    signal_type="correction",
                    confidence=0.8,
                    matched_text=match.group(0),
                    importance_boost=self.IMPORTANCE_BOOSTS["correction"],
                ))
                break  # one correction signal per text

        # Check emphasis patterns
        for pattern in self.EMPHASIS_PATTERNS:
            match = re.search(pattern, text, re.IGNORECASE)
            if match:
                signals.append(OrientingSignal(
                    signal_type="emphasis",
                    confidence=0.7,
                    matched_text=match.group(0),
                    importance_boost=self.IMPORTANCE_BOOSTS["emphasis"],
                ))
                break

        # Check error spike
        error_count = sum(
            1 for pattern in self.ERROR_INDICATORS
            if re.search(pattern, text)
        )
        if error_count >= self.ERROR_SPIKE_THRESHOLD:
            confidence = min(1.0, error_count / 5.0)  # 3->0.6, 5->1.0
            signals.append(OrientingSignal(
                signal_type="error_spike",
                confidence=confidence,
                matched_text=f"{error_count} error indicators",
                importance_boost=self.IMPORTANCE_BOOSTS["error_spike"],
            ))

        # Check pacing break
        if message_lengths and len(message_lengths) >= 2:
            recent = message_lengths[:-1]
            current = message_lengths[-1]
            avg = sum(recent) / len(recent)
            if avg > 0 and current < avg * self.PACING_BREAK_RATIO:
                signals.append(OrientingSignal(
                    signal_type="pacing_break",
                    confidence=0.6,
                    matched_text=f"message length {current} vs avg {avg:.0f}",
                    importance_boost=self.IMPORTANCE_BOOSTS["pacing_break"],
                ))

        # Compute overall importance boost (max, not sum — avoids over-boosting)
        importance_boost = max((s.importance_boost for s in signals), default=0.0)

        return OrientingResult(signals=signals, importance_boost=importance_boost)
    ```

    Key design decisions:
    - **Stateless**: detector takes text + optional message_lengths, no internal state. State management is the caller's responsibility.
    - **Case insensitive**: all pattern matching uses `re.IGNORECASE`.
    - **Word boundaries**: `\b` prevents false positives ("factually" doesn't match "actually").
    - **Max not sum for importance_boost**: a correction+emphasis text gets 0.3 (the correction boost), not 0.5. This keeps importance within reasonable range.
    - **No LLM calls**: pure re.search + arithmetic.
    - **Break after first match per category**: prevents double-counting when multiple patterns match in same category.
